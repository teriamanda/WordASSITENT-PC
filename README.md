# WordASSITENT-PC
这是一个专为电脑大屏设计的‘简单’背单词工具，是我一次次像ai描述完整需求后得出来的结果，虽然是个html轻量文件但胜在很方便，适合习惯边阅读边用 Excel 记录的人。它支持一键从表格同步【单词、翻译、例句】，核心逻辑是‘强制拼写 + 艾宾浩斯曲线’，通过手感和逻辑双重强化，解决在大屏阅读时单词记不住、记不深的问题，让复习变得既简单又精准。

💡 创作初衷
作为一名文科生，我深知在 PC 端进行深度阅读时，零散单词难以有效留存的痛点。虽然我不精通编程，但我善于发现需求并利用 AI 协作将创意落地。这款工具回归了最纯粹的“肌肉记忆”逻辑，解决大屏环境下单词记不住、复习没规律的问题。

✨ 核心亮点
1.Excel 无缝衔接：先将日常学习到的英语单词记录在网页在线excel表格中（推荐👸号learnandrecord上面很多类似文章）。然后直接从你的网页Excel 复制【单词、翻译、例句】三列数据粘贴即可导入词库，实现零摩擦的词库积累。词库也可以实时更新，如果有新的单词加入点击词库管理后更新，把原来删掉再重新粘贴全部新的词库进去就行
![工具预览图](./词库说明.png)
2.强制拼写模式：模仿 Google 翻译的极简界面，强制要求完整拼写输入，将视觉记忆转化为手感记忆。
3.艾宾浩斯智能调度：算法严格遵循遗忘曲线，根据你的拼写正误自动安排复习时间（从 5 分钟到 15 天不等）。
4.“Slay”终极必杀：独创 Slay 功能，对于彻底掌握的单词一键斩杀，永不出现，并记录在成就日志中。
5.谷歌翻译隐身界面（方便随时随场景隐身使用）

📖 使用指南
数据导入：点击“词库管理”，将你的 Excel 单词表（三列格式）直接粘贴进输入框并保存。
拼写练习：
系统会根据例句自动生成语境填空。
在输入框内拼写单词并按 Enter 键确认。
😈状态反馈：
认识 / Know & Skip：单词进入下一个更长的复习周期。
不认识/模糊：单词重置记忆阶段，增加复习频次。
Slay：单词彻底从复习池移除，进入“斩杀日志”。
